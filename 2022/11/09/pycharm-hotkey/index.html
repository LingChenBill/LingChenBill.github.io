pycharm Mac版-快捷键

一.mac键说明

  • ⌘ Command
  • ⇧ Shift
  • ⌥ Option
  • ⌃ Control
  • ↩︎ Return/Enter
  • ⌫ Delete
  • ⌦ 向前删除键（Fn+Delete）
  • ↑ 上箭头
  • ↓ 下箭头
  • ← 左箭头
  • → 右箭头
  • ⇞ Page Up（Fn+↑）
  • ⇟ Page Down（Fn+↓）
  • Home Fn + ←
  • End Fn + →
  • ⇥ 右制表符（Tab键）
  • ⇤ 左制表符（Shift+Tab）
  • ⎋ Escape (Esc)

二.编辑

1. 窗口操作
`⇧` Shift + F4 				: 开启代码独立窗口.
`⌃` Control + Tab 		: 在窗口间进行切换.
`⌘` Command	+ w				: 关闭当前标签窗口.
2. 工具
`⌘` Command	+ 1				: 切换到工程窗口.
`⌘` Command	+ e				: 当前文件弹出，打开最近使用的文件列表.
