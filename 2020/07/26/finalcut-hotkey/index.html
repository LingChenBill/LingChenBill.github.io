Finalcut-Pro快捷键记录

1、播放

j：后退（连续按，几倍速后退）
k：暂停
l：前进（连续按，几倍速前进）

2、编辑

i：入口点
o：出口点

使用i、l 和 o 将视频中的片段标记好后，该片段会被黄色框起来。

f：将标记的视频片段收藏，片段上方会出现绿色横框
双击：用鼠标双击收藏好的绿线，可以选中该收藏视频片段
del/back：拒绝选中的视频，选择工具栏中的[所有片段] -> [隐藏被拒绝的]，该视频不可见
Ctrl + c：显示所有视频片段
Ctrl + h：隐藏被拒绝的的视频片段
Ctrl + f：显示收藏的视频片段
u：撤销操作

Command + n：新建项目（时间线）
d：将选中的音频加入到项目（时间线）中
Shift + z：将音乐片段放大，填充整个时间线，好控制
电影片段按钮：更改片段在时间线中的外观

空格键：播放与暂停
m：音乐打点（一般在波峰位置打点）

双击打点：可以选择删除，删除打点
q：双击视频片段后，按q键可以将视频加入到时间线中
w：双击视频片段后，按w键可以将视频插入到时间线中
e：双击视频片段后，按e键可以将视频追加到时间线的末尾
d：双击视频片段后，按d键可以将视频覆盖到选择的时间点上
Command + k：视频加入关键词，便于查找，打过关键词的视频上方会有一条蓝色线

t：在时间线可以修剪视频（前后拖拉），与a键联合使用，选择与修剪
p: 位置工具，拖拉片段可以在时间线中产生黑场
Command + +/-：放大与缩小时间线
Shift + z：快速的将时间线缩放到大小合适
s：打开或关闭视频/音频实时预览（鼠标拖拉预览）
n：吸附，打点，拖拉视频片段时使用

# 剪辑开始点
Option/Alt + [

# 剪辑结束点
Option/Alt + ]

3、资源库

资源库是视频工作的场所。可以保存在本机，也可以保存在外置SSD上。
正在剪切的视频资源要放在本机，加快剪切视频流畅度。
已经完成的视频资源尽量放在外置SSD上。节省本机硬盘空间。

资源库中的 整合，可以将外置的视频整合到资源库中，方便资源库的整体移动。

4、视频格式

视频一般格式如下

帧率：每秒包含或者播放的照片数。

码率的计算公式：

码流 = 文件大小（兆位）/ 视频持续时间（秒）
例：200MB * 8 / 40 秒 = 40Mbps

5、关键帧

在视频的开始点和结束点可以加入关键帧，加入关键帧动画。

Ctrl + v：可以查看关键帧信息
关键帧可以配合效果实现不同的特效。
