Docker微服务镜像推送至阿里云

1.登录阿里云Docker Registry

前提:

  • 本机安装docker
  • 本机安装有maven

登录阿里云docker镜像

# 登录.
% docker login --username=your_name registry.cn-beijing.aliyuncs.com
# 输入服务密码.

2.打包微服务

# 在idea命令窗口.
% source ~/.bash_profile 
% mvn clean package -Dmaven.test.skip=true
......
 % ls target 
classes                                 eureka-0.0.1-SNAPSHOT.jar.original      maven-archiver
eureka-0.0.1-SNAPSHOT.jar               generated-sources                       maven-status

3.编写Dockerfile

在工程根目录下创建Dockerfile文件:

# 拉取jdk镜像.
FROM openjdk:8-alpine

# 添加微服务jar,并命名为app.jar.
ADD target/*.jar app.jar

# 暴露端口号.
EXPOSE 8761

# 运行jar.
ENTRYPOINT ["java", "-jar", "/app.jar"]

4.构建微服务docker镜像

# 镜像路径:registry.cn-beijing.aliyuncs.com/lcyun/
# 镜像名:eureka
% docker build -t registry.cn-beijing.aliyuncs.com/lcyun/eureka .
......
Successfully tagged registry.cn-beijing.aliyuncs.com/lcyun/eureka:latest

# 查看该镜像.
% docker images
REPOSITORY                                      TAG                 IMAGE ID            CREATED             SIZE
registry.cn-beijing.aliyuncs.com/lcyun/eureka   latest              7a8fa6abf26e        10 seconds ago      155MB

5.推送镜像

% docker push registry.cn-beijing.aliyuncs.com/lcyun/eureka

访问阿里云镜像仓库,该镜像已推送.

6.编写shell脚本

在工程根目录下创建build.sh文件:

#!/usr/bin/env bash

mvn clean package -Dmaven.test.skip=true

docker build -t registry.cn-beijing.aliyuncs.com/lcyun/eureka .

docker push registry.cn-beijing.aliyuncs.com/lcyun/eureka

启动脚本,一键打包,上传微服务镜像到阿里云仓库

% bash build.sh
docker aliyun micro
