Idea中scala配置

1.scala下载

访问官网下载

https://www.scala-lang.org/download/all.html

以版本:scala-2.12.11为例

解压到指定文件夹.

tar xf scala-2.12.11.tgz -C ~/Documents/software
% cd ~/Documents/software
% ls
scala-2.12.11

2.idea配置

idea -> preferences -> Plugins -> scala(搜索) -> 安装scala插件.

Installed -> 重启Idea.

3.项目配置scala

File -> Project Structure -> Global Libraries -> + -> scala sdk

也可以工程中进行如下操作进行添加scala sdk.

4.scala测试

创建hello spark测试scala

New -> Scala Class -> 文件名 + Object

package com.atg.bigdata.spark.core

/**
 * scala测试.
 *
 * @description:
 * @author: lingchen
 * @date: 2021/1/3
 */
object Test {

  def main(args: Array[String]): Unit = {
    print("Hello spark!")
  }
}

右击启动Test文件(Ctrl + Shift + R)

Hello spark!
Process finished with exit code 0
