第一个serverless程序

1.注册用户

进入阿里云云开发平台,登录用户.

https://workbench.aliyun.com

创建空白应用,本例以NodeJs和FC计算服务为例.

点击应用卡片上的开发部署按钮,进入云端CloudIDE.

2.编辑程序

查看readme.md,查看帮助
编辑helloworld.js

module.exports.handler = function(event, context, callback) {
    var request = JSON.parse(event);// 请求内容都会存储在 event 中，JSON编码后可遍历查看具体的内容
    // do sth
    var msg = '你好，世界！';
    // 返回值格式
    var responseObject = {
        isBase64Encoded: false,//与 body 内容是否进行 base64 编码保持一致
        statusCode: 200,//状态码，根据返回值自行决定适合的状态码
        headers: {
        	"Content-type": "text/html; charset=utf-8"//根据返回值设置正确的 Content-type
        },
        body: msg//返回值
    };
    callback(null, responseObject);//返回结果
}

3.编辑api接口

/api文件夹中的文件提供服务接口,如helloworld

  • data主体: msg
  • headers: “Content-type”: “application/json; charset=utf-8”
  • body: msg
module.exports.handler = function(event, context, callback) {
    var request = JSON.parse(event);// 请求内容都会存储在 event 中，JSON编码后可遍历查看具体的内容
    // do sth
    var msg = {
      "to": "世界",
      "from": "lingchen",
      "data": "你好啊!"
    };

    // 返回值格式
    var responseObject = {
        isBase64Encoded: false,//与 body 内容是否进行 base64 编码保持一致
        statusCode: 200,//状态码，根据返回值自行决定适合的状态码
        headers: {
        	"Content-type": "application/json; charset=utf-8"//根据返回值设置正确的 Content-type
        },
        body: msg//返回值
    };
    callback(null, responseObject);//返回结果
}

4.编辑index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
  </head>
  <body>

  <article>
    <em id="to"></em>
    <h1 id="message"></h1>
    <address id="from"></address>
  </article>

  <script type="text/javascript">
  fetch('/api/helloworld').then(function(response) {
    return response.json();
  }).then(function(msg) {
    console.log(msg)
    document.querySelector("#to").innerText = "美丽的: " + msg.to;
    document.querySelector("#message").innerText = msg.data;
    document.querySelector("#from").innerText = "热爱着你的: " + msg.from;
  })
  </script>
  </body>
</html>

5.测试

点击 CloudIDE 侧边栏 「WB」插件，打开「测试」标签页，在「用户路径」中输入我们刚刚创建的helloworld.js的访问地址 + /helloworld

https://test1yi9nuhejv6wza3cxh7.workbenchapi.com/helloworld

你好，世界！

访问地址:每次测试或者发布部署后都不一样.

6.部署

部署上线前，请务必配置到 serverless.js 文件的 SAFE 列表进行保护

const SAFE = [
  '/api',
  '/helloworld.js',
  '/.workbench',
  '/serverless.js'
];

打开「WB」插件的「部署」标签页，点击「部署」，会弹出部署信息确认，点击「继续部署」开始构建部署，请耐心等待，直到部署完成.

https://test1yi9nuhejv6wza4nran.workbenchapi.com/

7.停止服务

在应用卡片中,不用应用时,请停止服务,防止流量超过免费限制,产生费用.
