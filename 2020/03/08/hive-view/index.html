​ 视图处理：
​ 视图是一种用select语句定义的“虚表”。视图可以用来以一种不同于磁盘实际存储形式把数据呈现给用户。
​ 在Hive中，创建视图时并不把视图“物化”存储到磁盘上。相反，视图的select语句只是在执行引用视图的语句时才执行。

Hive视图命令

hive> create view valid_records
    > as
    > select *
    > from records2
    > where temperature != 9999
    >   and (quality == 0 or quality == 1 or quality == 4 or quality == 5 or quality == 9);

OK
Time taken: 0.445 seconds

hive> show tables;
OK
records2
valid_records
Time taken: 0.024 seconds, Fetched: 2 row(s)

​ 创建视图时并不执行查询，查询只是存储在metastore中。show tables命令的输出结果里包括视图。

基于视图创建视图

hive> create view max_temperatures (year, max_temperature)
    > as
    > select year, max(temperature) from valid_records
    > group by year;

OK
Time taken: 0.194 seconds

Hive子查询

​ 子查询是内嵌在另一个SQL语句中的SELECT语句。Hive对子查询的支持很有限。它只允许子查询出现在SELECT语句的FROM子句中。

hive> select year, avg(max_temperature)
    > from (
    >   select year, max(temperature) as max_temperature
    >   from records2
    >   where temperature != 9999
    >     and (quality == 0 or quality == 1 or quality == 4 or quality == 5 or quality == 9)
    >   group by year
    > ) mt
    > group by year;
