re-正则表达式-01

1.分隔空格,;

asdf fjdk; afed, fjek,asdf, foo

匹配符:

[;,\s]\s*

匹配结果:

line = 'asdf fjdk; afed, fjek,asdf, foo'
s = re.split(r'[;,\s]\s*', line)
print('re split: ', s)

结果:
re split:  ['asdf', 'fjdk', 'afed', 'fjek', 'asdf', 'foo']

函数 re.split() 是非常实用的，因为它允许你为分隔符指定多个正则模式.
比如， 在上面的例子中，分隔符可以是逗号，分号或者是空格，并且后面紧跟着任意个的空格.
只要这个模式被找到，那么匹配的分隔符两边的实体都会被当成是结果中的元素返回.

2.括号捕获分组

使用 re.split() 函数时候，需要特别注意的是正则表达式中是否包含一个括号捕获分组.
如果使用了捕获分组，那么被匹配的文本也将出现在结果列表中.

匹配符:

(;|,|\s)\s*

匹配结果:

line = 'asdf fjdk; afed, fjek,asdf, foo'
fields = re.split(r'(;|,|\s)\s*', line)
print('re split (): ', fields)

结果:
re split ():  ['asdf', ' ', 'fjdk', ';', 'afed', ',', 'fjek', ',', 'asdf', ',', 'foo']

3.非捕获分组

不想保留分割字符串到结果列表中去，但仍然需要使用到括号来分组正则表达式的话，
确保你的分组是非捕获分组，形如 (?:...)

匹配符:

(;|,|\s)\s*

匹配结果:

line = 'asdf fjdk; afed, fjek,asdf, foo'
a = re.split(r'(?:,|;|\s)\s*', line)
print('re split ?: ', a)

结果:
re split ?:  ['asdf', 'fjdk', 'afed', 'fjek', 'asdf', 'foo']

4.多行匹配

为了修正这个问题，你可以修改模式字符串，增加对换行的支持.
在这个模式中，(?:.|\n) 指定了一个非捕获组 (也就是它定义了一个仅仅用来做匹配，而不能通过单独捕获或者编号的组).

text2 = '''/* this is a
 multiline comment */ 
        '''

multi_comment = re.compile(r'/\*((?:.|\n)*?)\*/')
c = multi_comment.findall(text2)
print('findall multi lines found: ', c)

结果
findall multi lines found:  [' this is a\n multiline comment ']
