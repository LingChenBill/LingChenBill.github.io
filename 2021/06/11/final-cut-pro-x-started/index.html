Final-cut-pro-x-学习记录_01

1.导入素材

导入素材时,要将素材拷贝到资源库里,防止素材被删除后,资源库不能使用.

l: 播放素材
k: 暂停
j: 往后播放
空格: 播放/暂停
左/右键: 选中不同的素材(黄色边框线)

2.粗剪素材

l: 播放
i: 视频入点
k: 暂停
o: 视频出点
f: 加入收藏(视频上方出现绿色横条)
u: 撤销收藏
backspace: 删除素材(素材上出现黄线横线)
u: 撤销删除素材(黄线消失)

q: 连接 -> 将所选素材片段放置到当前时间轴所处的位置,不抢占任何片段的位置
w: 插入 -> 将所选素材片段插入到当前时间轴所处的位置,原有的片段会被自动切割开来
e: 附加 -> 将所选素材片段附加到当前时间轴最末尾的位置,不抢占任何片段的位置
d: 覆盖 -> 将所选素材片段覆盖到当前时间轴所处的位置,原有片段的位置不变,但内容会被裁切并覆盖

^ + f/ctrl + f: 素材只显示收藏过的素材片段
^ + c/ctrl + c: 素材显示所有的素材片段

3.精剪和磁性时间线

b: 切割 -> 对任何素材片段进行切割,方便对片段进行分割调整
a: 选择 -> 分割完成后,按a键回到正常的"选择工具"模式
Option/Alt + g: 将几个片段同时选定,然后右键"新建复合片段",可以几段片段变成一个普通片段
Command(cmd) + Option(Alt): 按住键,并拖动音频可以改变音频与视频的连接点
p + Option(Alt) + w: 按p键切换到[位置]选择时,按Option(Alt) + w,
										 就会在所选位置上插入一个占位的空白的素材,方便以后在此插入新的视频素材

4.多轨道剪辑

将多个片段重叠在一起进行剪辑

主序列上的片段是所有片段的核心
其它轨道上的片段都依靠着主序列片段
每一条片段都和主序列片段之间都有一条连接线,表示存在着连接与对应关系
在移动和删除主序列时片段时,要注意其它片段可能出现的移动和删除情况

Command(cmd) + Option(Alt) + 上: 将所选素材片段移动上主故事线上方
Command(cmd) + g: 将主故事线上方的素材转换成次故事线
Command(cmd) + Option(Alt) + 下: 将次故事线上的素材移到到主故事线上方
转场: 将主故事线上方的两个素材中间添加一个转场效果后,会将这两个素材转换成一个次故事线

5.关键帧和渐进运动

关键帧, 可以理解为片段中”做下标记”的那一帧
两个以上的关键帧才能互相配合使用,实现某种运动或变化
关键帧的作用就是记录片段属性,每个关键帧都记录一种属性

6.卡点视频

将音乐收藏素材插入时间线

shift + z: 扩展片段至合适的时间线
m: 音乐打点(双击可以删除mark点)
t: 可以在时间片段中左右拖动视频
慢速: 可以调节视频片段的播放速度
防抖动: 全选视频片段,可以去除视频的抖动

音频

Option(Alt): 在音频片段时,按Option(Alt),点击两个间隔的点,再将音频向下拉,可以做一个音频逐渐降低的效果.

7.调色

颜色版

  • 颜色
  • 饱和度
  • 曝光

可以拖动圆点,可以修改视频片段的显示效果

8.聚焦/抠像/遮罩

聚焦: 可以制造出一个整体模糊而局部清晰的效果
     从而实现某种程度的伪"景深"效果,弥补在画面拍摄中留下的遗憾与不足
     聚焦效果本身可以极大地提升画面的通透感,凸显画面主体,营造艺术性氛围
抠像: 一种抠像器,一种是高度抠像器
     抠像的本质是从画面中抠选出一部分,而将其它部分从画面中抹去
遮罩: 将一段视频遮罩一部分,凸显在下部的视频当中

8.转场/特效/文件输出

转场:
两段没有编辑过的视频素材中间添加[转场]时,会出现[片段边缘之外没有足够的额外媒体可用来创建转场]的弹框.
添加转场后,整个视频的时长会改变.
这是因为转场要替换掉一定的左右边素材的时长.编辑过的素材中间添加转场后,则不会出现些弹框.

双击转场效果中间的按钮: 可以查看精确度编辑器,可以方便的查看转场所占用的左右素材时长

光流: 视频效果 -> 光流(慢速视频可以设置)
分离音频: 选中视频片段 -> 分离音频(可以删除视频中声音,拍摄时可以有噪音)
转场: 可以在视频的开始与结尾添加合适的转场,可以让片段平滑地过渡,减少观看者的视觉不适感
添加字幕: 可以视频开始时添加合适的字幕(可以在字幕中添加转场)
输出: 可以输出为Apple设备720p或者Apple设备1080p(设备标签页中:H.264编码)
hotkey fcpx
