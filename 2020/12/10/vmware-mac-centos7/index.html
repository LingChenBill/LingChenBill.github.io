Mac-vmware中Centos7网络连接

1.虚拟机安装

前提

  • VMware Fusion安装
  • Centos7虚拟镜像

导入Centos7虚拟镜像,

右击 -> 网络适配器 -> 连接网络适配器 -> 与我的Mac共享

启动Centos7

# 用户名与密码.
% root
root

2.设置主机名

% hostnamectl set-hostname hdss7-11.host.com

# 退出重新登录.
% logout
root
root

# 主机名已变成hdss7-50.
[root@hdss7-50 ~]#

3.关闭防火墙

% sestatus
% vi /etc/selinux/config

SELINUX=disabled

# 重启.
% reboot

# 查看.
getenforce

# 关闭防火墙.
systemctl status firewalld
systemctl stop firewalld
systemctl disable firewalld.service

4.配置IP地址

查看mac本机的vmnet8的网关地址，这个对应NAT模式的网络连接.

% cd /Library/Preferences/VMware Fusion/vmnet8
# 查看和修改nat.conf
% cat nat.conf

# NAT gateway address
ip = 10.4.7.2
netmask = 255.255.255.0

修改虚拟机中的文件: ifcfg-ens33

[root@hdss7-50 ~]# vi /etc/sysconfig/network-scripts/ifcfg-ens33

TYPE=Ethernet
PROXY_METHOD=none
BROWSER_ONLY=no
BOOTPROTO=static
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_FAILURE_FATAL=no
IPV6_ADDR_GEN_MODE=stable-privacy
NAME=ens33
UUID=8d71fc45-b8dc-42c4-9f8c-65f17323eb5a
DEVICE=ens33
ONBOOT=yes
IPADDR=10.4.7.50
NETMASK=255.255.255.0
GATEWAY=10.4.7.2
DNS1=10.4.7.2
#DNS1=114.114.114.114

# 自己设置IP: IPADDR=10.4.7.50
# 修改如下内容:
# BOOTPROTO=static
# NETMASK=255.255.255.0
# GATEWAY=10.4.7.2
# DNS1=10.4.7.2

重启network服务

% service network restart
# 或者
% systemctl restart network.service

查看Mac中的wifi中的dns配置,并配置

wifi -> 打开网络偏好设置 -> 高级 -> DNS

# dns可以不设
10.4.7.11
192.168.0.253
192.168.1.1
