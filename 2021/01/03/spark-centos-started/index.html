Spark的local环境安装

1.环境准备

前提

  • macOS Big Sur 11.1(主机环境)
  • VMware Fusion12以上版本(与macOS Big Sur 11.1兼容)
  • centos7虚拟环境

2.jdk安装

$ tar -zxvf /opt/src/jdk-8u221-linux-x64.tar.gz -C /usr/local/soft/

配置JAVA_HOME环境变量

$ vi .bash_profile

export JAVA_HOME=/usr/local/soft/jdk1.8.0_221

PATH=$PATH:$HOME/bin

PATH=${JAVA_HOME}/bin:$PATH
export PATH

使jdk生效

source .bash_profile

# 查看版本.
$ java -version
java version "1.8.0_221"
Java(TM) SE Runtime Environment (build 1.8.0_221-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.221-b11, mixed mode)

3.spark安装

$ tar -zxvf src/spark-3.0.0-bin-hadoop3.2.tgz -C /opt/

# 建立软连接.
$ ln -s /opt/spark-3.0.0-bin-hadoop3.2/ /opt/spark-local

4.spark程序验证

建立数据文件

$ vi /opt/datas/word.txt 
Hello spark
Hello scala
Hello spark
Hello scala
Hello spark

在spark的命令行中输入:

// 测试spark是否可用.
scala> var i = 10
i: Int = 10

// 统计word.txt中的单词个数.
scala> sc.textFile("/opt/datas/word.txt").flatMap(_.split(" ")).map((_,1)).reduceByKey(_+_).collect
res0: Array[(String, Int)] = Array((scala,2), (Hello,5), (spark,3))
