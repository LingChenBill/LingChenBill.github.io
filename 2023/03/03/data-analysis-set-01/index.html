2.数据

数据是对事物的描述和记录.

  • 定类数据: 按照类别属性进行分类, 各有类别之间是平等并列关系. 这种数据不带数量信息, 并且不能在各类别间进行排序.
    主要数值运算, 计算每一类别中的项目的频数和频率.
  • 定序数据: 定序数据之间可以进行排序, 比较优劣. 通过将编码进行排序, 可以表示之间的高低差异.
  • 定距数据: 具有一定单位的实际测量值; 定距数据的精确性比定类数据和定序数据更高; 可以计算出各变量之间的实际差距(加, 减).
  • 定比数据: 可以比较大小, 进行加, 减, 乘, 除运算; 定距尺度中, 0表示数值, 定比尺度中, 0表示”没有”; 定比数据中是存在绝对零点的, 而定距数据不存在.
  • 定性数据: 是一组表示事物性质, 规定事物类别的文字表述型数据.
    定类数据 + 定序数据
  • 定量数据: 指以数量形式存在着的属性, 并因此可以对其进行测量.
    定距数据 + 定比数据

3.统计指标

体现总体数量特征的概念和数值.
根据数据分析的目的不同, 统计指标也会变化.

总量指标: 特定条件下的总规模, 总水平或工作总量; 是一种最基本的统计指标.

平均指标: 用一个数字显示其一般水平; 集中趋势指标.

相对指标: 两个有联系的现象数值相比得到的比率; 描述的是相对关系, 而不是总体情况.

比例: 各数据 / 总比%

比率: 数据项 : 数据项

倍数: 突出上升, 增长幅度

环比, 同比:
$$
(本期数 - 同期数) / 同期数 * 100%
$$
同比增长率
环比更注重短期的涨幅表现.
同比理注重长期的.

4.集中趋势指标

用于体现数据一般水平的指标.
最快速了解样本数据的概况.
最常用的集中趋势指标就是平均值.
$$
平均值 = \dfrac{所有数据相加}{数据的个数}
$$
中位数: 按顺序排列后, 居于中间位置的数.
奇数: 位于(n + 1) / 2 位置的数.
偶数: 最中间的两位数相加 / 2

众数: 出现次数最多的数值; 反应的是局部特征, 密集度; 众数可以有多个.

5.离散趋势指标

离散趋势指标: 波动大小, 体现内部差异度的指标

  • 极差: 相距最远的两个点之间的距离. 数据内部最大的差异情况. 并不能真正的体现数据内部的离散程度.

  • 平均差: 一组数据与平均值差异的平均差异
    $$
    \dfrac{|每个数据项 - 均值|的总和}{数据项个数}
    $$
    数据项与平均值的差距越大

    数据越分散, 反之越集中

  • 标准差

    事件驱动型的数据, 在样本量较小的时候, 容易导致误差.

    对离散值更敏感.

$$
\sqrt[2]{\dfrac{|每个数据项 - 均值 |^2的总和}{数据项个数}}
$$

​ 优化过的更能代表离散程度的指标.

7.识别异常值

与平均值偏差极大或极小的值, 也叫离群点.

观察检测值与整体数据的差异度.

计算与平均值的倍数.
更多的检测方法: Tableau 和 Python.

判定异常值:

  • 错误记录
  • 错误添加

正确, 真实的异常值 -> 是否反映特殊事件 -> 修改, 调整
根据实际情况调整: 数值 * 需调整的比率.

同期性的数据 -> 不做处理

错误数据的处理:

  • 填充空值
  • 填充样本平均值

8.数据分析的流程

目标确定 -> 数据获取 -> 数据清洗 -> 数据整理 -> 描述分析 -> 洞察结论 -> 撰写报告

预测性分析

数据获取:

  • 字段设计
  • 数据提取

数据清洗:

异常值 : 识别, 判定, 处理
空白值, 无效值, 重复值: 修正 平均值填补, 删除

数据整理:

格式化: 日期的处理, 行列格式化
指标计算

描述分析:

数据描述: 数据的基本情况
指标统计: 分析实际情况的数据指标

洞察结论:

数据报告的核心
体现数据分析能力

撰写报告:

报告背景 -> 业务现状
报告目的 -> 解决什么问题
数据基本情况 -> 数据可信度
可视化图表 -> 数据的事理解程度
策略选择 -> 提出解决方案
