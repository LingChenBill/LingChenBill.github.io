jenkins-linux环境搭建

jenkins安装

下载地址：

https://pkg.jenkins.io/redhat-stable/

安装jenkins：

$ rpm -ivh jenkins-2.164.3-1.1.noarch.rpm

查看jenkins默认安装路径：

$ rpm -ql jenkins

[root@VM_0_16_centos ~]# rpm -ql jenkins
/etc/init.d/jenkins
/etc/logrotate.d/jenkins
/etc/sysconfig/jenkins
/usr/lib/jenkins
/usr/lib/jenkins/jenkins.war
/usr/sbin/rcjenkins
/var/cache/jenkins
/var/lib/jenkins
/var/log/jenkins

jenkins启动相关配置

​ 修改jenkins默认工作目录：

$ vim /etc/sysconfig/jenkins

# JENKINS_HOME="/var/lib/jenkins"
JENKINS_HOME="/home/jenkins/jenkins"

​ 修改jenkins默认用户以及端口号：

#JENKINS_USER="jenkins"
JENKINS_USER="root"

# Port Jenkins is listening on.
# Set to -1 to disable
#
JENKINS_PORT="8190"

​ 修改jenkins所需要的jdk：

$ vim /etc/init.d/jenkins

# see http://www.nabble.com/guinea-pigs-wanted-----Hudson-RPM-for-RedHat-Linux-td25673707.html
candidates="
/usr/local/soft/jdk1.8.0_212/bin/java
/etc/alternatives/java
/usr/lib/jvm/java-1.8.0/bin/java
/usr/lib/jvm/jre-1.8.0/bin/java
/usr/lib/jvm/java-1.7.0/bin/java
/usr/lib/jvm/jre-1.7.0/bin/java
/usr/bin/java
"

​ 启动jenkins：

$ service jenkins start

访问jenkins

浏览器访问

http://152.136.104.216:8190/

​ 解锁jenkins：

$ cat /home/jenkins/jenkins/secrets/initialAdminPassword

​ 第一次解锁后，设置jenkins登录用户与密码（牢记！）

​ 第一次安装必须插件过慢时，停止jenkins服务，进入jenkins工作目录，修改jenkins镜像源

$ cd /home/jenkins/jenkins/updates

sed -i 's/http:\/\/updates.jenkins-ci.org\/download/https:\/\/mirrors.tuna.tsinghua.edu.cn\/jenkins/g' default.json && sed -i 's/http:\/\/www.google.com/https:\/\/www.baidu.com/g' default.json

​ 然后启动jenkins服务，进入jenkins后，手动安装插件：

​ 如：系统管理 -> 插件管理 -> 过滤搜索：git

jenkins必要工具配置

​ 安装git：

$ yum install git

$ whereis git
