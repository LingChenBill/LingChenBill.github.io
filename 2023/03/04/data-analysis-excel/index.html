4.数学函数

数值 -> 数学函数 统计计算

数学函数: 是对数值类型数据进行计算, 数学换算, 统计, 类型转换.

基本换算:

  • 平方根

    =sqrt()

  • 幂运算

    =power

  • 总和

    =sum

  • 平均值

    =average

  • 求整

    =int

  • 求最近的偶数

    =even

集中趋势指标:

平均值: 总和/数据项个数.

=AVERAGE(H3:H14)

中位数: 从小到大排序后位于中间的数.

=MEDIAN(H3:H14)

众数: 出现次数最多的数据项.

=MODE(H3:H14)

离散趋势指标:

极差: 最大值 - 最小值

=MAX(H3:H14)-MIN(H3:H14)

平均差: 每个数据项与平均值差异程度的总和 / 数据项个数.

=AVEDEV(H3:H14)

标准差: 每个数据项与平均值差异程度的总和的平方 / 数据项个数.

=STDEV(H3:H14)

8.逻辑函数

能明确用「是否」或「真假」逻辑来回答.

逻辑运算: 与, 或, 非.

FALSE: 返回逻辑值FALSE
TRUE: 返回逻辑值TRUE
IF: 判断真假并返回对应值
AND: 交集运算
OR: 并集运算
NOT: 求反运算

逻辑函数对推论进行真假判断的函数.

IF函数 > 对不同的判断结果进行数值匹配.

对时间列(如:10:23) 进行规则判断:

=IF(INT(LEFT(K3, 2)) < 10, "早上", IF(INT(LEFT(K3, 2)) < 12, "上午", "下午"))

判断岗位是否为数据分析师, 是 - 输出"目标岗位", 否 - 输出空值

=IF(C25="数据分析师","目标岗位","")

9.条件聚合函数

在符合特定条件下的数据项进行统计: 数学函数 + 逻辑函数

SUMIF(S): 针对符合条件的数据项**”求和”**.

COUNTIF(S): 针对符合条件的数据项**”计次”**.

总数 > SUM函数

SUM函数: 点选符合条件的项目, 太慢, 每次都需要点选

SUMIF函数: 条件聚合函数
在公式中写死条件, 原始数据顺序随意.

统计地区的个数:
=COUNTIF(C:C, D2)

筛选一线城市的个数:
=SUM(SUMIFS(E:E,D:D,{"北京","上海","广州","深圳"}))

统计总职位数:
=COUNTIF(I:I, "*") - 1

统计"*"开发工程师:
=COUNTIF(I:I,"*开发工程师")

自动化的报表功能: 汇总, 统计, 排序, 数据的实时刷新.

耗时几分钟
自动化数据汇总

处理表格时间 > 1小时 -> 数据透视表

全选数据区域 -> 插入Tab -> 数据透视表 -> 选择要存放的位置 -> 选择对统计的行, 列, 统计列
为了方便统计结果, 可以选择列: 开始Tab -> 条件格式 -> 数据条 -> 选择样式

更改数据原数据内容后, 单击数据透视表后, 在数据透视表分析Tab中, 点击刷新按钮, 即可刷新最新的数据.

针对行标签中的日期类型的数据, 可以更改日期筛选频率, 点击行标签中的行, 右击选择组合, 更改 分组 中的设置即可.
